### Порождающие паттерны:

* Абстрактная фабрика (Abstract Factory)
* Строитель (Builder)
* Фабричный метод (Factory Method)
* Прототип (Prototype)
* Одиночка (Singleton)

#### other:
* Ленивая инициализация (lazy initialization)
* Объектный пул (object pool)
* Пул одиночек (Multiton)
